# Homepage

This is my personal website.
A small static site showcasing me and my projects.

Stack:
 - Express
 - Pug
 - Tailwind

# Usage

**Env variables**

- `PORT` (default: `8080`)
- `HOST` (default: `0.0.0.0`)

**For development**

```bash
npm run watch
node server.js
```

**For production**

```bash
npm run build
node server.js
```
